## Tech Used
* MVVM
* RxSwift
* Firebase Firestore & Storage & Auth
* MapKit
* CoreLocation
* SDWebImage
* Programmatic Constraints
* UIKit with no Storyboard
* Custom TableView & NavigationController & Tab Bar
